Sobre Mim


Olá, Sou Allan Felipe, acadêmico de Ciência da Computação pela Uninter e entusiasta de computação e infraestrutura de TI.


Minha jornada na TI é movida pela curiosidade de entender como o software interage com hardware. Atualmente, foco meus estudos na programação, desenvolvimento de sistemas robustos em Java e na análise técnica de suporte em ambientes de missão crítica. Tenho facilidade em diagnosticar falhas em redes IP e arquiteturas de sistemas distribuídos, buscando otimizar a performance de infraestruturas de TI e a segurança de dados.

Meu Portfólio


No meu portfólio, eu não foquei apenas na interface. Desenvolvi uma estrutura em Javascript que simula o comportamento de um sistema distribuído, com barras de carregamento que representam uma latência de uma rede IP e o tempo de resposta de um servidor em Java com um fluxo de dados do cliente ao servidor.


Projetos desenvolvidos com foco em soluções de suporte e eficiência de processamento. Aqui você encontrará aplicações que unem o código Java com a agilidade do desenvolvimento Web, sempre priorizando o monitoramento de performance e a facilidade de suporte técnico em TI.


Cada projeto aqui seguiu um levantamento de requisitos e testes, garantindo que o software suporte as demandas de uma rede de sistemas real.
